### Atv 4

Nesse programa, utilizamos a classe Scanner para obter a entrada do usuário.

variáveis numero1, numero2 e numero3. Em seguida, calculamos a 
média geométrica dos três números utilizando a fórmula:

Por fim, exibimos a média geométrica na tela utilizando System.out.println().
